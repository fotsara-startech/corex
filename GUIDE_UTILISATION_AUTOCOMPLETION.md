# Guide d'utilisation - Recherche de clients avec autocomplétion

## Nouvelle fonctionnalité: Recherche intelligente

Lors de la création d'une nouvelle collecte de colis, la recherche de clients (expéditeur et destinataire) est maintenant beaucoup plus simple et rapide!

## Comment ça marche?

### Étape 1: Commencez à taper

Dans le champ de recherche, commencez simplement à taper:
- Le nom du client
- Son numéro de téléphone
- Son adresse email

**Important**: Tapez au moins 2 caractères pour voir les suggestions.

### Étape 2: Voyez les suggestions

Dès que vous avez tapé 2 caractères, une liste de suggestions apparaît automatiquement sous le champ:

```
┌─────────────────────────────────────────────┐
│ 🔍 Tapez le nom, téléphone ou email...     │
└─────────────────────────────────────────────┘
  ┌───────────────────────────────────────────┐
  │ 👤 Jean Dupont                            │
  │    0612345678                             │
  │    Abidjan - Cocody, Rue des Jardins      │
  ├───────────────────────────────────────────┤
  │ 👤 Jeanne Martin                          │
  │    0623456789                             │
  │    Abidjan - Plateau, Avenue 12           │
  └───────────────────────────────────────────┘
```

### Étape 3: Sélectionnez le bon client

Cliquez simplement sur le client que vous cherchez dans la liste.

### Étape 4: Vérifiez les informations

Tous les champs sont automatiquement remplis:
- ✅ Nom complet
- ✅ Téléphone
- ✅ Ville
- ✅ Adresse
- ✅ Email (si disponible)

Un message de confirmation apparaît: "Client sélectionné: [Nom du client]"

## Exemples d'utilisation

### Exemple 1: Recherche par nom

**Vous tapez**: "Kou"

**Suggestions affichées**:
- Kouadio Yao - 0712345678
- Kouassi Marie - 0723456789
- Kouman Jean - 0734567890

**Vous cliquez** sur "Kouadio Yao" → Tous les champs sont remplis!

### Exemple 2: Recherche par téléphone

**Vous tapez**: "071"

**Suggestions affichées**:
- Tous les clients dont le téléphone commence par "071"

### Exemple 3: Recherche par email

**Vous tapez**: "jean@"

**Suggestions affichées**:
- Tous les clients avec un email contenant "jean@"

## Astuces

### 💡 Astuce 1: Effacer la recherche
Cliquez sur le bouton "X" à droite du champ pour effacer et recommencer.

### 💡 Astuce 2: Client non trouvé?
Si aucune suggestion n'apparaît, le client n'existe pas encore. Remplissez simplement les champs manuellement, et le client sera créé automatiquement.

### 💡 Astuce 3: Beaucoup de résultats?
Si la liste est longue, vous pouvez:
- Scroller dans la liste avec la molette de la souris
- Taper plus de caractères pour affiner la recherche

### 💡 Astuce 4: Navigation au clavier
Vous pouvez utiliser:
- ↑ ↓ (flèches haut/bas) pour naviguer dans les suggestions
- Entrée pour sélectionner
- Échap pour fermer les suggestions

## Avantages

### ⚡ Plus rapide
- Pas besoin de cliquer sur "Rechercher"
- Résultats instantanés pendant que vous tapez

### 👁️ Plus clair
- Vous voyez toutes les informations du client
- Pas de popup supplémentaire

### 🎯 Plus précis
- Recherche dans tous les champs (nom, téléphone, email)
- Affinage automatique en tapant plus de caractères

### 🖱️ Moins de clics
- 1 seul clic pour sélectionner
- Pas de bouton "Rechercher" à cliquer

## Cas particuliers

### Cas 1: Homonymes
Si deux clients ont le même nom:
- Regardez le téléphone pour les différencier
- Regardez l'adresse
- Cliquez sur le bon client

### Cas 2: Numéro de téléphone similaire
Si plusieurs clients ont des numéros proches:
- Tapez plus de chiffres pour affiner
- Regardez le nom pour confirmer

### Cas 3: Aucun résultat
Si aucune suggestion n'apparaît:
- Vérifiez l'orthographe
- Essayez avec le téléphone
- Si le client n'existe vraiment pas, remplissez manuellement

## Comparaison: Avant vs Après

### ❌ Avant (ancienne méthode)
1. Taper le nom dans le champ
2. Cliquer sur "Rechercher"
3. Attendre le chargement
4. Si plusieurs résultats, une popup s'ouvre
5. Choisir dans la popup
6. Cliquer sur "Valider"

**Total: 4-5 clics, plusieurs secondes**

### ✅ Maintenant (nouvelle méthode)
1. Commencer à taper
2. Cliquer sur le bon client dans les suggestions

**Total: 1 clic, instantané!**

## Questions fréquentes

### Q: Pourquoi rien ne s'affiche quand je tape 1 caractère?
**R**: Pour éviter trop de résultats, la recherche commence à partir de 2 caractères.

### Q: Puis-je toujours créer un nouveau client?
**R**: Oui! Si le client n'existe pas, remplissez simplement les champs manuellement. Le client sera créé automatiquement lors de la validation.

### Q: La recherche fonctionne-t-elle hors ligne?
**R**: Non, la recherche nécessite une connexion pour interroger la base de données.

### Q: Puis-je modifier les informations après sélection?
**R**: Oui! Après avoir sélectionné un client, vous pouvez modifier n'importe quel champ si nécessaire.

### Q: Les suggestions sont-elles limitées?
**R**: Oui, pour des raisons de performance, un maximum de résultats est affiché. Si vous ne trouvez pas votre client, tapez plus de caractères pour affiner.
